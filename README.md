# spring-boot-redis

Redis - SpringBoot integration

## Getting Started

### Redis:run

```
docker-compose up --build
```

### Project Build

```
mvn clean
mvn install 
```

### Project Run

```
mvn spring-boot:run
```

### Test

```
localhost:8080/people
localhost:8080/messages
```
